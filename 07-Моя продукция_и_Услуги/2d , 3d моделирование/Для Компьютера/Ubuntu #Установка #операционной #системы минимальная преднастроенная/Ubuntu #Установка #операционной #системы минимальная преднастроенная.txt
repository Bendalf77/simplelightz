#Установка операционной системы(ОС) минимальная перенастроенная.2 000 ₽
Перенастроенная означает: 
1).  Русский язык ввода с переключением на английский одной кнопкой;
2).  Русский язык интерфейса и имена папок файлов, содержимое txt документов;
3).  Шрифты с русским языком минимальный набор из 2-5шт вместо 1500шт по умолчанию - экономия озу и времени места;
4).  Чистая ОС с xfce4 или mate   ;
4.1).  службы - нет печати и драйверов печати, нет общего доступа, ничего нет  ;
4.2).  программы - нет браузера. нет драйверов, нет ничего. есть только проводник  ;
5). Удалены локализации кроме русской и словари шрифты кроме русского, английского с учётом п1-3 ;
6).  назначены другие шрифты вместо по умолчанию везде тк было сделано п1-5   ;
7).  Консоль системная - интерпретатор bash без плагинов вместо sh, ввод русских и английских символов   ;
8).  Можешь распаковать любой формат архива, запаковать в zip, 7z, tar, bz, gz ;
9).Всё в один раздел на диске при разметке в mbr. 2 раздела при uefi, gpt.
10). Пароль root отсутствует, системные политики по умолчанию, есть 1 пользователь с правами администратора, фаервола нет;
  Дополнительные платные услуги:  смотри в этом разделе и в "перевозки", "консультации"
Работа выполнена с должным качеством когда возможно войти в систему и пользоваться пп1-10.
